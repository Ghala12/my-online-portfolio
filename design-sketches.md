# Design Sketches - Portfolio Website

## Mobile View
```
Header: Navigation Menu
↓
Hero: Name & Title  
↓
Projects: Vertical List
↓
About: Text Section
↓
Contact: Info
↓
Footer
```

## Desktop View
```
Header: Logo | Navigation Links
↓
Hero: Large Banner
↓
Projects: 3-Column Grid
↓
About & Contact: Side by Side  
↓
Footer
```

## Design Notes:
- Mobile: Single column for touch-friendly use
- Desktop: Multi-column for better space usage
- Responsive breakpoints at 768px and 992px
- Dark mode support for accessibility
